AI Native Best Practices

5 Rules for Maximum Output

  • Treat context like RAM — every token costs quality, money, and speed
  • /clear between tasks — the #1 missed optimization. Without it, task 5 has only ~20% context
  • Use subagents for research — they get their own 200K context window. Route to Haiku ($0.80/M)
  • Hooks for deterministic savings — RTK hook, test filter, auto-format. Unlike CLAUDE.md, hooks always run
  • Skills over CLAUDE.md bloat — skills load on demand. CLAUDE.md loads every session. Target: under 200 lines
